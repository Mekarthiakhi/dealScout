# DealScout AI

AI-powered scalable shopping comparison platform.

## Stack
- React + Vite + TypeScript
- Tailwind + ShadCN
- Node.js + Express
- MongoDB + Redis
- Puppeteer Scrapers
- Docker Ready

## Run

### Frontend
```bash
cd apps/web
npm install
npm run dev
```

### Backend
```bash
cd apps/api
npm install
npm run dev
```
